# 更新日志 v1.0.1

## 🐛 Bug修复

### 修复的问题
1. **修复了 `titleSpan.textContent` 的 null 引用错误**
   - 在更新浮窗内容前添加元素存在性检查
   - 优化了浮窗创建和内容更新的时序

2. **优化了高亮功能的错误处理**
   - 添加了完整的 try-catch 错误捕获
   - 避免在特殊元素（input、textarea等）中进行高亮
   - 改进了跨元素选择的处理逻辑

3. **改进了浮窗显示逻辑**
   - 使用 setTimeout 确保 DOM 完全创建后再更新内容
   - 添加了更多的调试日志

## 🔍 如何验证修复

1. 打开浏览器控制台 (F12)
2. 刷新页面
3. 选择文本
4. 应该不再看到 "Cannot set properties of null" 错误

## 📝 使用建议

- 如果之前安装过插件，建议先卸载再安装新版本
- 安装后刷新所有已打开的网页
- 在测试页面 (test.html) 上先进行测试
