# 英文划词收集器 Chrome 插件

一个轻量级的Chrome浏览器插件,帮助你在阅读英文网站时快速收集和整理重要的词句。

## ✨ 功能特性

- 🖱️ **划词高亮**: 鼠标选择文本后自动高亮显示
- 📝 **实时收集**: 选中的词句自动添加到收集列表
- 💾 **持久化存储**: 所有收集的内容本地保存,关闭浏览器不丢失
- 🎨 **美观浮窗**: 优雅的浮动窗口展示收集内容
- 🚀 **拖拽移动**: 浮窗可自由拖动到任意位置
- 📥 **导出功能**: 一键导出所有收集的词句为txt文件
- 🗑️ **管理功能**: 支持单个删除和批量清空

## 📦 安装步骤

### 方法一: 开发者模式加载

1. 下载或克隆此项目到本地
2. 打开Chrome浏览器,访问 `chrome://extensions/`
3. 在右上角开启 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择项目文件夹 `word-collector-extension`
6. 安装完成! 你会在浏览器右上角看到插件图标

### 方法二: 打包为crx文件(可选)

1. 在 `chrome://extensions/` 页面
2. 点击 **"打包扩展程序"**
3. 选择插件根目录
4. 生成 `.crx` 文件后可分享给他人安装

## 🎯 使用方法

### 基本使用

1. **划选文本收集**
   - 在任意英文网站上用鼠标选中你想收集的词句
   - 文本会自动高亮显示(黄色背景)
   - 同时自动添加到收集列表

2. **查看收集列表**
   - 划选文本后浮窗会自动显示在右上角
   - 也可以点击浏览器工具栏的插件图标,选择"显示浮窗"

3. **管理收集内容**
   - **拖动浮窗**: 点击并拖动浮窗顶部可移动位置
   - **最小化**: 点击 ➖ 按钮折叠浮窗
   - **关闭浮窗**: 点击 ✖️ 按钮关闭(不会删除内容)
   - **删除单条**: 点击每条记录右侧的"删除"按钮
   - **清空全部**: 点击 🗑️ 按钮清空所有收集

4. **导出数据**
   - 点击浮窗中的 📥 按钮
   - 或点击插件图标,选择"导出全部词句"
   - 自动下载为txt文件,包含文本、来源URL和时间

## ⚙️ 配置说明

### 修改高亮颜色

编辑 `content.css` 文件中的 `.word-collector-highlight` 样式:

```css
.word-collector-highlight {
  background-color: #ffeb3b;  /* 修改为你喜欢的颜色 */
}
```

### 修改浮窗位置

编辑 `content.css` 文件中的 `.word-collector-float` 样式:

```css
.word-collector-float {
  top: 60px;    /* 修改顶部距离 */
  right: 20px;  /* 修改右侧距离 */
}
```

## 🐛 常见问题

**Q: 为什么有些网站无法使用?**  
A: 部分网站(如Chrome Web Store、chrome://等内部页面)出于安全限制无法注入内容脚本。

**Q: 点击按钮没反应怎么办？**  
A: 请查看详细的 [故障排查指南](TROUBLESHOOTING.md)

**Q: 数据存储在哪里?**  
A: 所有数据使用Chrome Storage API存储在本地浏览器中,不会上传到任何服务器。

**Q: 如何备份我的收集数据?**  
A: 使用导出功能将数据保存为txt文件,或者定期备份Chrome的用户数据目录。

**Q: 高亮在刷新后消失了?**  
A: 高亮是临时的DOM标记,页面刷新后会消失,但收集的内容已保存,不会丢失。

**Q: 清空后页面高亮还在？**
A: 请确保使用最新版本，清空后刷新页面或使用浮窗中的清空按钮。

⚠️ **重要提示：安装或更新插件后，必须刷新页面才能使用！**
